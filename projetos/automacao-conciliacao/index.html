Desafios e Soluções

  • Problema: Dados bancários em formatos inconsistentes (CSV, PDF, APIs).
    Solução: Desenvolvimento de um parser universal com Python e Pandas.
  • Problema: Lentidão na consulta de transações históricas.
    Solução: Criação de um cache em banco SQLite com indexação inteligente.

Arquitetura

  • Extrator: Robô Python/Selenium para coletar dados
  • Transformador: Scripts SQL/Pandas para tratamento
  • Loader: Power BI para visualização
